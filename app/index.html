Available endpoints:

JMeter

  • GET /ok : returns 200
  • GET /report : perform jmeter test with defaults (threads:100, ramp:60, duration:120) and generate html report
  • POST /report : same as GET, accepts following params in body - threads, ramp, duration
  • GET /reports/list : list existing reports

Newman CLI

  • GET /newman : get previous html report
  • GET /newman/report?env=LT.postman_environment.json&colleciton=Fx1.postman_collection.json : runs newman CLI-generated report, based on postman tests collection for specified environment
  • GET /newman/upload : simple upload form for an array of json files - newman collections and environments
  • GET /newman/settings/list : List of uploaded newman collection and environment settings (json)
